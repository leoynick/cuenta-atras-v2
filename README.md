**## Usage**

**### Install Dependencies**

```

# Install Node

$ cd cuenta-atras-v2

$ npm install

```

**### Use it**

```

# development mode

$ npm run dev

# production build for Mac

$ npm run build:mac

# production build for Windows

$ npm run build:win64

```
